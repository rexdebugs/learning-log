WSL stands for Windows Subsystem for Linux.
It lets you run a Linux environment (like Ubuntu) directly on Windows.
WSL2 uses a lightweight virtual machine to improve compatibility and performance.
It helps developers use Linux tools while still using Windows apps like VS Code.
With the VS Code WSL extension, you can edit Linux files and run Linux commands easily
