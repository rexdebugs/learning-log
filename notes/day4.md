## What these symbols mean

- `~` means my home directory
- `.` means my current directory
- `..` means the parent directory, one level up

## Absolute path
An absolute path is the full address of a file or folder starting from `/`.

## Relative path
A relative path is an address based on where I am currently standing in the terminal.
